Тема 6: Функции

Цель: Научиться создавать и использовать функции в C++ через управление Кодиксом, выполнение повторяющихся действий и добавление логики для работы с картами и сигналами.
Задание 1: Функция для подсчёта сигналов

Описание: Создайте функцию countSignals(), которая подсчитывает количество сигналов на текущей карте. Функция должна возвращать это количество.

Требования:

    Функция принимает параметры: размеры карты (mapWidth, mapHeight).
    Функция использует вложенные циклы для проверки каждой клетки карты.
    Возвращает количество сигналов.

Пример вызова:

int totalSignals = countSignals(3, 3);
cout << "Общее количество сигналов на карте: " << totalSignals << endl;

Пример работы программы:

Общее количество сигналов на карте: 3

Задание 2: Функция для перемещения Кодикса

Описание: Напишите функцию moveCodix(int steps), которая перемещает Кодикса вперёд на указанное количество шагов. Если перед Кодиксом есть препятствие, он прекращает движение.

Требования:

    Функция принимает параметр steps – количество шагов.
    Кодикс делает шаги с помощью step(), пока это возможно.
    Если перед Кодиксом препятствие, движение останавливается.

Пример вызова:

moveCodix(5);

Пример работы программы:

Кодикс сделал 3 шага и остановился перед препятствием.

Задание 3: Функция для сбора сигналов

Описание: Напишите функцию collectAllSignals(), которая перемещает Кодикса по всей карте и собирает все сигналы. Функция должна выводить общее количество собранных сигналов.

Требования:

    Функция принимает параметры: размеры карты (mapWidth, mapHeight).
    Использует вложенные циклы для обхода карты.
    Проверяет и подбирает сигналы на каждой клетке.

Пример вызова:

collectAllSignals(3, 3);

Пример работы программы:

Сигнал подобран на позиции (0, 1).
Сигнал подобран на позиции (2, 2).
Общее количество подобранных сигналов: 2
