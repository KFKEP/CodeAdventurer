Тема 4: Условные операторы

Цель: Научиться использовать условные операторы (if, else if, else) через проверку состояния карты, препятствий и сигналов для управления действиями Кодикса.
Задание 1: Проверка препятствия перед Кодиксом

Описание: Используйте функцию frontCheck() для проверки, есть ли препятствие перед Кодиксом. Если препятствие есть, выведите сообщение: "Путь заблокирован". Если препятствий нет, выполните шаг вперёд с помощью функции step().

Требования:

    Проверить наличие препятствия перед Кодиксом.
    Если путь свободен, переместить Кодикса вперёд.
    Если путь заблокирован, вывести сообщение.

Пример работы программы:

Путь свободен, Кодикс делает шаг вперёд.

или

Путь заблокирован, Кодикс не может двигаться вперёд.

Подсказка:

    Используйте if (frontCheck() == 0) для проверки свободного пути.

Задание 2: Проверка сигнала под Кодиксом

Описание: Используйте функцию checkForBeep() для проверки, стоит ли Кодикс на сигнале. Если сигнал есть, подберите его с помощью функции collectBeep(). Если сигнала нет, выведите сообщение.

Требования:

    Проверить наличие сигнала под Кодиксом.
    Если сигнал есть, подобрать его и вывести сообщение: "Сигнал подобран".
    Если сигнала нет, вывести сообщение: "Сигнал отсутствует".

Пример работы программы:

Сигнал подобран.

или

Сигнал отсутствует.

Подсказка:

    Используйте if (checkForBeep() == 1) для проверки наличия сигнала.

Задание 3: Решение лабиринта

Описание: Кодикс должен двигаться вперёд, пока путь свободен. Если путь блокирован, он поворачивает направо и продолжает движение. Напишите программу, которая использует условные операторы для управления поведением Кодикса в лабиринте.

Требования:

    Используйте frontCheck() для проверки пути.
    Если путь свободен, выполнить шаг вперёд.
    Если путь заблокирован, повернуть направо с помощью turnRight().
    Завершить выполнение после 10 шагов.

Пример работы программы:

Шаг 1: Кодикс двигается вперёд.
Шаг 2: Путь заблокирован, поворот направо.
Шаг 3: Кодикс двигается вперёд.
...

Подсказка:

    Используйте цикл для повторения действий, пока не выполнено 10 шагов.
