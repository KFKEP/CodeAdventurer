Тема 2: Переменные и типы данных

Цель: Научиться использовать переменные и основные типы данных C++ через управление Кодиксом и взаимодействие с картой.
Задание 1: Переменные для координат Кодикса

Описание: Создайте переменные для хранения текущих координат Кодикса на карте и выведите их в консоль. Переместите Кодикса на несколько позиций, обновите координаты и снова выведите их.

Требования:

    Используйте переменные типа int для хранения координат x и y.
    Используйте функцию placePlayerOnMap() для размещения Кодикса.
    Выведите координаты до и после перемещения.

Пример работы программы:

Текущие координаты Кодикса: (1, 1)
Кодикс перемещён на позицию: (2, 3)

Подсказка:

    Координаты можно менять вручную или вводить через пользовательский ввод.
    Используйте placePlayerOnMap() для обновления позиции Кодикса.

Задание 2: Вычисление доступных шагов

Описание: Считайте размер карты и вычислите максимальное количество шагов, которые может сделать Кодикс, если он будет двигаться по прямой до края карты.

Требования:

    Используйте переменные для хранения размеров карты (mapWidth и mapHeight).
    Вычислите количество шагов до границы карты в любом направлении.
    Выведите результат в консоль.

Пример работы программы:

Размер карты: 5x5
Кодикс может сделать максимум 4 шага вправо или вниз.

Подсказка:

    Размер карты можно установить вручную или считать из массива карты.

Задание 3: Типы данных для свойств Кодикса

Описание: Создайте переменные для хранения свойств Кодикса:

    Энергия (int): от 0 до 100.
    Название карты (string).
    Находится ли на сигнале (bool).

Используйте эти переменные в программе и отобразите их значения в консоли.

Требования:

    Создайте переменные для каждого свойства.
    Используйте функции checkForBeep() для проверки сигнала и обновления переменной bool.
    Выведите текущие свойства Кодикса.

Пример работы программы:

Текущие свойства Кодикса:
Энергия: 100
Название карты: map1
На сигнале: Да

Подсказка:

    Используйте true/false для переменной bool.
    Обновляйте значения переменных по мере выполнения программы.
