Тема 3: Арифметические операции

Цель: Ознакомиться с основными арифметическими операциями в C++ через работу с перемещением Кодикса, подсчётом шагов и изменением карты.
Задание 1: Подсчёт оставшихся шагов

Описание: Кодикс начинает движение с определённым количеством энергии, где каждый шаг тратит 10 единиц энергии. Напишите программу, которая подсчитывает, сколько шагов может сделать Кодикс до полной потери энергии.

Требования:

    Создайте переменную energy (энергия) с начальным значением 100.
    Рассчитывайте оставшиеся шаги на основе энергии.
    Выведите в консоль количество возможных шагов.

Пример работы программы:

Энергия Кодикса: 100
Кодикс может сделать максимум 10 шагов.

Подсказка:

    Используйте оператор деления / для подсчёта количества шагов.

Задание 2: Подсчёт сигналов на карте

Описание: Используйте арифметические операции для подсчёта общего количества сигналов (B) на текущей карте.

Требования:

    Используйте функцию checkForBeep() для проверки наличия сигнала под Кодиксом.
    Проходите по каждой клетке карты, перемещая Кодикса, и считайте сигналы.
    Выведите итоговое количество сигналов.

Пример работы программы:

Общее количество сигналов на карте: 3

Подсказка:

    Используйте цикл для перемещения Кодикса по карте.
    Инкрементируйте счётчик при обнаружении сигнала.

Задание 3: Вычисление координат следующей клетки

Описание: Напишите программу, которая вычисляет координаты следующей клетки, на которую должен перейти Кодикс. Используйте арифметические операции для этого расчёта.

Требования:

    Используйте переменные для хранения текущих координат (x, y).
    Рассчитайте новые координаты на основе текущего направления движения.
    Выведите координаты следующей клетки.

Пример работы программы:

Текущая позиция Кодикса: (1, 1)
Следующая клетка: (1, 2)

Подсказка:

    Используйте арифметические операции (+ или -) в зависимости от направления движения.
