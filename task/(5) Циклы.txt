Тема 5: Циклы

Цель: Освоить использование циклов в C++ (for, while, do-while) через управление Кодиксом и выполнение повторяющихся действий, включая работу с сигналами (beep).
Задание 1: Подбор всех сигналов на карте

Описание: Используйте цикл для перемещения Кодикса по карте и подбора всех сигналов. Кодикс должен проверять наличие сигнала на каждой клетке и подбирать его, если он есть.

Требования:

    Кодикс должен последовательно обходить каждую клетку карты.
    Если на клетке есть сигнал, он должен быть подобран с помощью collectBeep().
    В конце работы выведите общее количество подобранных сигналов.

Пример работы программы:

Сигнал подобран на позиции (0, 1).
Сигнал подобран на позиции (2, 2).
Общее количество подобранных сигналов: 2

Подсказка:

    Используйте вложенные циклы для обхода карты.
    Координаты клеток можно обновлять через перемещение Кодикса с помощью placePlayerOnMap().

Задание 2: Сбор сигналов в строке

Описание: Кодикс должен двигаться вдоль строки карты и подбирать все сигналы, встречающиеся на пути. Если путь заблокирован, Кодикс завершает выполнение.

Требования:

    Используйте цикл while для перемещения Кодикса вправо до конца строки.
    На каждой клетке проверяйте наличие сигнала и подбирайте его, если он есть.
    Если путь заблокирован (frontCheck() == 1), завершите выполнение.

Пример работы программы:

Сигнал подобран на позиции (0, 1).
Сигнал подобран на позиции (0, 3).
Кодикс завершил движение.

Подсказка:

    После проверки наличия сигнала перемещайте Кодикса с помощью step().

Задание 3: Сбор сигналов в лабиринте

Описание: Кодикс должен двигаться по лабиринту, пока не подберёт все сигналы. Если путь вперёд блокирован, он поворачивает направо. После подбора всех сигналов программа завершает выполнение.

Требования:

    Используйте цикл do-while для управления движением Кодикса.
    На каждой клетке проверяйте наличие сигнала и подбирайте его.
    Если путь заблокирован, поворачивайте направо с помощью turnRight().
    Программа завершает выполнение, когда все сигналы подобраны.

Пример работы программы:

Сигнал подобран на позиции (0, 1).
Сигнал подобран на позиции (1, 2).
Сигнал подобран на позиции (2, 3).
Все сигналы подобраны. Кодикс завершил движение.

Подсказка:

    Для подсчёта оставшихся сигналов можно использовать отдельную переменную-счётчик.
