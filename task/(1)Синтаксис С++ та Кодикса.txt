Тема 1: Синтаксис и структура программы C++

Цель: Ознакомить ученика с базовым синтаксисом C++ через использование функций Кодикса для работы с картами.
Задание 1: Отобразить карту

Описание: Используя функции setMap() и rendering(), отобразите карту в консоли. Кодикс по умолчанию работает с картой map1, но нужно явно указать карту через функцию setMap().

Требования:

    Установить карту map1 с помощью setMap().
    Отобразить её с помощью rendering().

Пример работы программы:

Карта загружена:
> . . .
. X . .
. . . B

Подсказка:

    Используйте функцию setMap() для выбора карты.
    После выбора карты вызовите rendering().

Задание 2: Выбрать и отобразить карту

Описание: Программа должна запросить у пользователя название карты и отобразить её. Если карта не найдена, вывести сообщение об ошибке.

Требования:

    Запросить у пользователя название карты.
    Установить карту с помощью setMap(). Если карта существует, отобразить её с помощью rendering().
    Реализовать проверку: если карта не существует, вывести сообщение об ошибке.

Пример работы программы:

Введите название карты: map2
Карта map2:
> X . .
. . B .
. . . .

Подсказка:

    Проверить корректность ввода можно с помощью простого поиска карты в папке maps.

Задание 3: Вывести карту с координатами Кодикса

Описание: Используя функции placePlayerOnMap() и displayMap(), добавьте Кодикса на карту на заданные координаты. Затем отобразите карту с помощью displayMap().

Требования:

    Установить карту с помощью setMap().
    Разместить Кодикса на карте на позицию (1,1).
    Отобразить карту с помощью displayMap().

Пример работы программы:

Карта с Кодиксом:
К . . .
. X . .
. . . B

Подсказка:

    Для размещения Кодикса используйте функцию placePlayerOnMap().

Пример кода для первого задания:

#include <iostream>
#include "codeadventurer.h" // предположим, что это основной заголовочный файл Кодикса
using namespace std;

int main() {
    // Устанавливаем карту
    setMap("map1");

    // Отображаем карту
    rendering();

    return 0;
}

Пример кода для третьего задания:

#include <iostream>
#include "codeadventurer.h"
using namespace std;

int main() {
    // Устанавливаем карту
    setMap("map1");

    // Размещаем Кодикса на позиции (1, 1)
    placePlayerOnMap(1, 1);

    // Отображаем карту
    displayMap();

    return 0;
}
